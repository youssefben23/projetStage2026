# Plateforme de Gestion de Modèles d'E-mails - Frontend Angular

## Installation

```bash
npm install
```

## Démarrage

```bash
npm start
```

L'application sera accessible sur `http://localhost:4200`

## Structure du Projet

- `src/app/components/` - Composants Angular
- `src/app/services/` - Services (API, Auth)
- `src/app/models/` - Modèles TypeScript
- `src/app/guards/` - Guards de routage
- `src/app/interceptors/` - Intercepteurs HTTP

## Fonctionnalités

- Authentification (Login/Register)
- Liste des templates
- Éditeur HTML/CSS avec aperçu en temps réel
- Gestion des versions
- Validation HTML/CSS
- Recherche de templates
